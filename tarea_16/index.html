Este pagina cuenta con los siguientes comandos por voz:

  • azul: cambia el color a azul del texto de donde se edita el texto
  • rojo: cambia el color a rojo del texto de donde se edita el texto
  • negro: cambia el color a negro del texto de donde se edita el texto

Y las siguientes palabaras para efectos en el microfono por voz:

  • Rebota
  • Sorpresa
  • Salta
  • Sigilo
  • limpia -> para quitar los efectos
